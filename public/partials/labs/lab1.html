SVN

In this course we are using SVN to submit all our assignments

Once you are in your VM please checkout your svn repo with the following command 
svn co https://subversion.ews.illinois.edu/svn/fa15-cs241/YOUR-NETID cs241
which will check out your entire SVN repo into a folder called 'cs241' into your current directory. Now change your directory into the 'cs241' folder 
cd cs241
and create a 'Lab1' folder 
mkdir Lab1
Now check the status of your repo 
svn status
and notice that you now have a 'Lab1' folder that needs to be added. Add the 'Lab1' folder to svn 
svn add Lab1
and commit it 
svn ci -m "adding the Lab1 folder"

Valgrind + GDB + GCC

In this course you will need to know how to use Valgrind + GDB + GCC

Once you are in your VM please create a file called 'dumb.c' in your Lab1 directory that you just made. Now write a valid C program in 'dumb.c' that will segfault. Now compile with gcc 
gcc dumb.c -o dumb -g
Run it to make sure it segfaults 
./dumb
Run valgrind to see what line it segfaults on 
 valgrind ./dumb
Run gdb to see what line it segfaults on 
gdb ./dumb 
run
backtrace
Then exit gdb 
quit
