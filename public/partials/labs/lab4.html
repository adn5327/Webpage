Review a Bad Text Editor

Your lab instructor will go over a bad text editor.

You must fill out code_review.txt. Note that you must both fill out this code review AND attend lab to get full points. If you fail to do either than you will recieve a 0. Fill out the text file in the following format. 1 line per good coding practice that the bad text editor could have used or was shown in the good text editor. All lines should start with the line number (1 indexed) followed by a closing parenthesis followed by a single space followed by the non empty string that contains your answer followed by a newline character (not a newline escape sequence). There should be at least 5 lines.
