cat

In this lab you will be implementing 'cat'.
cat - concatenate files and print on the standard output. So if a user enters 
cat hello.txt world.txt
then cat will output all the lines of hello.txt followed by world.txt. So if hello.txt contained 
hello
and world.txt contained 
world
then 
cat hello.txt world.txt
would print out 
hello
world

sort

In this lab you will be implementing 'sort'.
sort - sort lines of text files So if a user enters 
sort file1.txt file2.txt
then sort will output all the lines of file1.txt file2.txt in sorted order. So if file1.txt contained 
3.14
pi
and file2.txt contained 
e
2.17
then 
sort file1.txt file2.txt
would print out 
2.17
3.14
e
pi

time

In this lab you will be implementing 'time'.
time - run programs and reports how long they took So if a user enters 
time ./prog1 --- ./prog2
Notice that the programs are seperated by " --- ". then time will run prog1 THEN prog2 and record how long it took in seconds. The format should be the amount of time it took in seconds out to the 6th decimal place + " seconds". Note that we only care about wall clock time.

date

In this lab you will be implementing 'date'.
For this one you should match exactly what the default date does. 
man date

wc

In this lab you will be implementing 'wc'.
wc - print newline, word, and byte counts for each file. Just follow the format of the actual wc. 
man wc

ediff

c

2
3

>    1
