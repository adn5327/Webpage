Note:
The tricky_step problem in part2 attempts to allow for computations with the "+" and the "-" operators with different data types (Strings, Integers and Floats)

It expects some implicit conversion between types such as:

1) FLOAT +/- INT = FLOAT ( 10.0 - 2 = 8.0 )
2) STRING + INT/FLOAT = STRING ( "CS" + 241 = "CS241" - Appending a integer/float to a string )
3) STRING - INT/FLOAT = ERROR ( "CS241" - 241, ERROR is one of the enumerated types in part2-functions.h )

You may also make the following assumptions:

1) Any float or integer in the problem will not be longer than 4 digits.
2) When a float is appended to a string, it is truncated to 2 decimal places.

Before attempting the tricky_step problem, you will need to have a solid understanding of function pointers, structs and unions.

Here are some resources that would help you understand them better:

Function Pointers - http://c.learncodethehardway.org/book/ex18.html
Structs - http://www.tutorialspoint.com/ansi_c/c_structured_datatypes.htm
Unions - http://www.tutorialspoint.com/cprogramming/c_unions.htm

Also feel free to review Angrave's crowd-sourced book
